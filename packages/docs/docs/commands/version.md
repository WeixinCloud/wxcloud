# version

使用 `wxcloud version` 可以管理云托管版本。

## version:list

查看版本列表

### 基本用法

```bash:no-line-numbers
wxcloud version:list
```

### 命令行参数

```bash:no-line-numbers
OPTIONS
  -e, --envId=envId
  -h, --help                     show CLI help
  -p, --page=page
  -s, --serviceName=serviceName
  --json                         是否以json格式展示结果

```
